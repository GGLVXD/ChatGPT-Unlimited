# ChatGPT-Unlimited
Generate response to questions using AI Chat.

## How To Use


```sh
const Chat = require("chatgpt-unlimited");

async function chat (){

const prompt = "Hello! How Are you?";

const response = await Chat.create(prompt);
console.log(response);

};

chat();
```
